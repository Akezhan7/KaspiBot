## Технические детали:

### SQL запросы:

**Получение продавцов с подсчетом:**
```sql
SELECT 
    s.*,
    COUNT(ps.product_id) as product_count
FROM sellers s
LEFT JOIN product_sellers ps ON s.merchant_id = ps.seller_id 
    AND ps.is_active = 1
GROUP BY s.merchant_id
HAVING product_count > 0
ORDER BY product_count DESC, s.merchant_name ASC
```

**Получение товаров продавца:**
```sql
SELECT 
    p.master_sku as product_id,
    p.title,
    p.url,
    ps.price,
    ps.last_seen
FROM product_sellers ps
JOIN products p ON ps.product_id = p.master_sku
WHERE ps.seller_id = ? AND ps.is_active = 1
ORDER BY ps.last_seen DESC
```

### Callback данные:

- `sellers_page_{N}` - навигация по страницам продавцов
- `seller_{merchant_id}` - открыть конкретного продавца

## Что работает после каждого скана:

1. **Новый продавец найден** → добавляется в `sellers` → будет в списке
2. **Продавец ушел с товара** → `is_active=0` → исчезает из списка
3. **Продавец вернулся** → `is_active=1` → снова в списке + уведомление
4. **Количество товаров** → автоматически пересчитывается при открытии списка
