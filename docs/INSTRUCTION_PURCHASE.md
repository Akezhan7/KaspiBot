# Инструкция: Контрольная закупка

> Для ответственного лица (Улшат и другие назначенные сотрудники)

---

## Что такое контрольная закупка?

Когда продавец-нарушитель не реагирует на предупреждения (WARN1 и WARN2),
система создаёт **юридическую заявку**. Чтобы подготовить дело для суда,
нужно совершить **контрольную закупку** — реально купить товар у нарушителя
и зафиксировать все данные.

---

## Пошаговая инструкция

### 1. Вы получите уведомление в Telegram

```
📦 Вам назначена контрольная закупка!
Заявка: #15
Продавец: ТОО "Рога и Копыта"
```

Это значит, что администратор назначил вас ответственным командой:
```
/assign_purchase 15 @your_username
```

### 2. Совершите покупку

- Зайдите на Kaspi.kz
- Найдите товар нарушителя (информация в уведомлении)
- Оформите и оплатите заказ
- **Сохраните**:
  - Номер заказа Kaspi
  - Чек / скриншот оплаты
  - Фото полученного товара
  - БИН/ИИН продавца (из чека или карточки магазина)

### 3. Введите данные в бота

Отправьте команду:
```
/purchase_done 15
```
(где `15` — номер заявки из уведомления)

Бот задаст 4 вопроса по очереди:

#### Шаг 1: БИН/ИИН продавца
```
Бот: Введите БИН/ИИН продавца:
Вы:  770412345678
```

#### Шаг 2: Номер заказа Kaspi
```
Бот: Введите номер заказа Kaspi:
Вы:  KZ-1234-5678-9012
```

#### Шаг 3: Документы
```
Бот: Отправьте фото/документы закупки (чеки, скриншоты):
Вы:  [отправляете фото чека]
Вы:  [отправляете фото товара]
Вы:  готово
```

> Отправляйте фото по одному. Когда все загружены, напишите **готово**.

#### Шаг 4: Комментарий
```
Бот: Введите комментарий (или «пропустить»):
Вы:  Товар получен 15.03, упаковка вскрыта
```

### 4. Подтверждение

Бот покажет введённые данные и попросит подтвердить:

```
Бот: ✅ Данные закупки:
     БИН/ИИН: 770412345678
     Заказ: KZ-1234-5678-9012
     Документов: 2
     Комментарий: Товар получен 15.03...

     Всё верно? (да/нет)
Вы:  да
```

### 5. Готово

После подтверждения данные сохраняются, и статус заявки меняется на
**«Готов к подаче иска»**. Юристы получат уведомление.

## Частые вопросы

**Что если я ошиблась при вводе?**
Напишите `/cancel_purchase` в любой момент диалога — ввод отменится.
Затем начните заново: `/purchase_done 15`

**Можно ли загрузить документы позже?**
Нет, все данные вводятся в рамках одного диалога. Подготовьте фото заранее.

**Какой формат БИН/ИИН?**
12 цифр без пробелов и дефисов: `770412345678`

**Сколько фото можно загрузить?**
До 10 файлов (фото или документы).
